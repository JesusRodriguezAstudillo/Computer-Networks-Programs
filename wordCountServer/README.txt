In order to compile the server program, use the line
	gcc wcserver.c -o wcserver

Execute the server program using the line
	./wcserver port_number
	
In order to compile the client program, use the line
	gcc wcclient.c -o wcclient

Execute the client program using the line
	./wcclient port_number
	
In order to test the program, you may type strings when prompted in the client program
the server will then send all the information back to the user.
